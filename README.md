# Fantasy Change Engine

> **Každé fantasy rozhodnutí má mít důvod.**

Fantasy Change Engine je transparentní rozhodovací nástroj pro fantasy manažery.

Jeho cílem není předpovědět každý výsledek. Pomáhá dělat nejlepší možná rozhodnutí podle informací dostupných v okamžiku uzávěrky.

## První produkt: Uklidňovátko

Uživatel vloží svou sestavu a dostane:

* hodnocení pravděpodobnosti startu hráčů,
* doporučení základní sestavy,
* pořadí lavičky,
* doporučeného kapitána,
* hlavní rizika,
* stručné a srozumitelné zdůvodnění.

Výstup bude mít tři vrstvy:

1. **Uklidnění** – jednoduchý verdikt.
2. **Důvody** – stručné vysvětlení.
3. **Engine** – detailní data, výpočty a předpoklady modelu.

## Principy

* Každé doporučení musí mít důvod.
* Nikdy nepředstíráme větší jistotu, než skutečně máme.
* Oddělujeme ověřená fakta od modelových odhadů.
* Nejdřív klid, potom analýza.
* Uživatel si sám volí hloubku vysvětlení.
* Dobré rozhodnutí neposuzujeme pouze podle výsledku.
